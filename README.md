# Playwright Python SauceDemo Automation

## Overview
Automation framework for validating login, product sorting, cart, and checkout flows
using Playwright with Python and POM design.

## Application Under Test
https://www.saucedemo.com

## Tech Stack
- Playwright (Python)
- Pytest
- Page Object Model (POM)

## Test Scenarios Covered
- Role-based login validation
- Product sorting (price and name)
- Cart add and remove functionality
- Complete checkout flow

## Framework Design
- Page Object Model
- Centralised locators
- Reusable utilities

## How to Run
pip install -r requirements.txt  
playwright install  
pytest

## Why This Project
Demonstrates end-to-end automation and role-based validation,
commonly asked in QA interviews.
